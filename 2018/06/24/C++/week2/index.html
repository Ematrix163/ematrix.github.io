函数指针

程序运行期间，每个函数都会占用一个段连续的内存空间。函数名就是该函数所占内存区域的起始地址。将函数的入口地址赋给一个指针变量，使该指针变量指向该函数。然后可以通过指针变量调用这个函数。这种指向函数的指针就被称为“函数指针”。

Q： 为什么要“多此一举”引入函数指针的概念？
A： e.g. C语言快速排序库函数，可以对任意类型的数组进行排序

比较函数编写规则：

  • 如果 *elem1 在前面， 函数的返回值为负整数；
  • 如果都行， 返回0
  • 如果 *elem2在前面，返回正整数

命令行参数

  • argc: 代表启动程序时，命令行参数的个数。可执行程序本身的文件名也算一个命令行参数，因次argc至少为1.
  • argv： 指针数组，每个元素都是一个char* 类型的指针，该指针指向一个字符串。
    例如，argv[0]指向的字符串就是第一个命令行参数，即可执行程序的文件名，argb[1]指向第二个命令行参数….

引用

某个变量的引用等价于这个变量的别名。即改变引用的值会影响原值。

特点：

  • 定义引用时一定要将其初始化成引用某个变量
  • 初始化后,它就一直引用该变量,不会再引用别的变量了
  • 引用只能引用变量,不能引用常量和表达式

动态内存分配

C语言中是用malloc实现的，C++中使用new运算符

用new运算符实现动态内存分配

第一种用法，分配一个变量

  • T是任意类型名，P是类型位T * 的指针。
  • 作用：动态分配出一片内存大小为sizeof(T)字节的内存空间,并且将该内存空间的其实地址赋值给P.

第二种方法,分配一个数组

作用：动态分配出一片内存大小为N*sizeof(T)字节的内存空间,并且将该内存空间的其实地址赋值给P.
