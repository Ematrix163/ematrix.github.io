AVL树节点个数和高度关系推导

发表于 2018-06-24 | 分类于 数据结构和算法
字数统计: 412 | 阅读时长 ≈ 2

根据已知AVL树的节点个数N来求解树的高度h。

问题提出

如果已经知道AVL树的树的节点个数为n，求问该数的最大树高是多少？

你可能会脱口而出是 log(n)， 数量级上确实是 log(n)，但是实际上这个答案还不够准确。

问题解决

因为根据题目要求，已知节点数为n，求最大树高。即最坏情况下的树高，因此我们可以认为最坏情况是每个AVL数的左右子树高差均为1，如图 1- 1所示：

图 1 - 1

我们可以得到如下递推式：

当前树的节点总个数 = 左子节点树的个数 + 右子节点树的个数 + 1

即:

​ $$S(h) = S(h - 1) + S (h - 2) + 1$$
稍作变形，得：

​ $$S(h) + 1 = (S(h - 1) - 1) + (S (h - 2) - 1)$$

显然S(h) + 1是一个斐波那契数列 (Fibonacci number)，因此只需要确定S(h) + 1数列的前两项，我们就可以知道h与S(h)的定量关系，如表1所示：

​ 表1 初始节点个数

h n T(h)
0 1 2 = fib(3)
1 2 3 = fib(4)

由表可我们可以知: $$S(h) + 1 = fib(h + 3)$$ 由此可得:

​ $$S(h) = \frac{\sqrt{5}}{5}*[(\frac{1+\sqrt{5}}{2})^{h+3} - (\frac{1-\sqrt{5}}{2})^{h+3}] - 1$$

略去高阶无穷小量，我们可以得到：
​ $$S(h) ≈ \frac{\sqrt{5}}{5}*(\frac{1+\sqrt{5}}{2})^{h+3} - 1$$
化简得：
​ $$h =\frac{log\sqrt{5}{S(h)}}{log\frac{1+\sqrt{5}}{2}} - 3$$ $$h = 1.44 log\sqrt{5} S(h) $$

注： 此处log指以2为底。

所以结论：

已知AVL树的树的节点个数为n，求问该数的最大树高为1.44log(n).
